# Get System Info

获取 Node.js 运行时环境信息。

## 参数

| 参数   | 类型   | 必填 | 说明                                            |
|--------|--------|------|------------------------------------------------|
| detail | string | 否   | 查询维度：`all` / `memory` / `platform` / `uptime`，默认 `all` |

## 使用场景

- 用户询问"系统内存还剩多少？"
- 需要确认 Node.js 版本兼容性
- 调试环境相关问题

## 示例

```json
{ "detail": "memory" }
// → "堆内存：45.2 MB / 128 MB（已用 35.3%）"

{ "detail": "all" }
// → 完整系统报告
```
