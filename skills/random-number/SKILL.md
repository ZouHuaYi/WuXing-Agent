# Random Number

生成指定范围内的随机整数。

## 参数

| 参数 | 类型    | 必填 | 说明            |
|------|---------|------|-----------------|
| min  | integer | 否   | 最小值（含），默认 1   |
| max  | integer | 否   | 最大值（含），默认 100 |

## 使用场景

- "帮我随机选一个 1~10 的数字"
- "模拟掷一个六面骰"
- 随机分配任务或顺序

## 示例

```json
{ "min": 1, "max": 6 }
// → "🎲 随机数（1~6）：4"
```
