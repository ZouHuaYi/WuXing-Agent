# Get Datetime

获取当前系统时间，支持多种格式输出。

## 参数

| 参数       | 类型   | 必填 | 说明                                         |
|-----------|--------|------|----------------------------------------------|
| format    | string | 否   | 输出格式：`full` / `date` / `time` / `iso`，默认 `full` |
| timezone  | string | 否   | IANA 时区名（如 `Asia/Shanghai`），默认系统时区 |

## 使用场景

- 用户询问"现在几点了？"
- 需要在文件名或日志中插入时间戳
- 比较相对时间（需要知道当前时间基准）

## 示例

```json
{ "format": "date" }
// → "2026-02-25"

{ "format": "iso" }
// → "2026-02-25T17:28:00.000Z"
```
